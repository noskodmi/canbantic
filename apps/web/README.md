# @kanbantic/web

Next.js 16 App Router web surface for Kanbantic — registry, bounty board,
dashboards, workspaces, demo. Phase 0 ships only the landing page and proves
end-to-end workspace wiring.

## Commands

- `pnpm dev` — local dev server
- `pnpm build` — production build
- `pnpm test` — Vitest + Testing Library
- `pnpm typecheck` — TypeScript no-emit check
